# GrafanaCon 2026 - AI Inception Workshop

## Getting Started

1. Wait for the Codespace to finish loading (Claude Code is being installed).
2. Run the setup script:
   ```
   bash setup.sh
   ```
3. Enter the workshop password (shared by the instructor).
4. Start Claude Code:
   ```
   claude
   ```
